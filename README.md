# Чат-бот «Отважный маркетинг» на базе Qwen (Yandex AI Studio)

Интеллектуальная диалоговая система для автоматизации консультирования клиентов и подбора услуг в малом маркетинговом консалтинге.

## О проекте

Бот помогает потенциальным клиентам:
- Получать информацию об услугах и ценах
- Проходить первичную квалификацию
- Записываться на консультацию

Разработан в рамках выпускной квалификационной работы по направлению **02.03.02 Фундаментальная информатика и информационные технологии** (профиль «Программная инженерия»).

## Основные возможности

- **RAG** (Retrieval-Augmented Generation) на основе документов агентства
- **Контекст диалога** (память последних сообщений)
- **Пошаговая запись на консультацию** через FSM
- **Шифрование персональных данных** (соответствие ФЗ-152)
- **Админ-панель** для сотрудников (просмотр и обработка заявок)
- **Постоянная клавиатура** + анимация «Думаю...»
- **Сохранение всей истории переписок** в базу данных

## Технологический стек

- **Python 3.12**
- **aiogram 3** — асинхронный фреймворк для Telegram
- **Qwen** (Yandex AI Studio) — генеративная модель
- **SQLAlchemy + AsyncPG** — работа с PostgreSQL
- **Cryptography** — шифрование персональных данных
- **FSM** (Finite State Machine) — управление состояниями диалога

## Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/descooly/qwen-telegram-bot.git
cd qwen-telegram-bot
```

### 2. Установка зависимостей
```bash
python -m venv .venv
source .venv/bin/activate    # для Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
```bash
cp .env.example .env
```

Заполните `.env` своими данными:
- `TELEGRAM_BOT_TOKEN`
- `YANDEX_FOLDER_ID`
- `YANDEX_API_KEY`
- `ADMIN_PASSWORD`

### 4. Запуск бота
```bash
python main.py
```
